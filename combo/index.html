Visualization: Combo Chart

Overview

The combo chart is a vertical columns combined with a single line chart rendered in the browser using SVG and HTML. Like all Crag Charts, combo charts display tooltips when the user hovers over the series elements. The chart is also limited to 20 data points and data will automatically be truncated if there are more points provided.

Unlike the column and line chart, each data point in the chart requires 2 values to each label, the first value is for the left side axis, the column, and the second is for the right side axis, the line.

The minimum required code required for creating a column chart is the following

<div id="combo-chart" style="width: 100%; height: 500px;"></div>

<script src="https://combinatronics.com/Xn1ch1/Crag-Charts/main/crag/crag.charts.loader.js"></script>
<script>
	let chart;
	new CragLoader(createCharts, ['combo']);

	function createCharts() {
		const options = {
			data: [['A', 500, 26], ['B', 369, 30], ['C', 420, 20], ['D', 466, 36]],
		}
		chart = new CragCombo(options).create('combo-chart');
	}
</script>

In the second script tag, the onLoadCallBack is invoked, the function paramater passed should be below this callback that will create the chart. For more detailed information, see How to Use.

Built In Functions

Title Update

The chart title can be updated at any time even if one was not defined in the initial options, should the chart need adjusting for the new space the _draw function will be called automatically. The title property should be applied to the chart object.

chart.title = 'New Chart Title';

Data Update

To update the data in the chart, the update function should be called on the chart object. Only one parameter is required, an array of arrays. See data parameter for more information. The data set can change in size when updated, bars and line points will be automatically added and removed as needed but will still be limited to a total of 20 points and truncated automatically. In addition, this function will automatically call the _draw function (below) once the update has been completed.

let newData = [['A', 160, 100], ['B', 527, 75], ['C', 428, 90]];
chart.update(newData);

Draw

The _draw function will cause a complete redraw of the chart with the pre existing data. This function will be useful where the size of the container object changes size. The below example is calling the _draw function on the chart object when the window resizes.

window.onresize = function() {
	chart._draw();
}

On Click Callback

A function can be passed to the onClick property in the options to the columns elements. This function will be called when a series columns is clicked.

let options = {
	columns: {
		onClick: myFunction
	}
}
myFunction(obj) {
	... ...
}

The function should have one parameter, an object which will be returned by the click listener. This object will contain information on the series point clicked in addition to the elements related to it. The object will contain the following

Object Parameters
columns The html element of the columns clicked as an object. Type: HTMLElement
label The html element of the label related to the columns clicked as an object. If the label positions are none, this will return null. Type: HTMLElement
name The name of the data point, this is the same name used in the hAxis. Type: String
point The html element of the point, related to the columns clicked, as an object. Type: HTMLElement
texts The formatted strings of the data point value. The format can be defined by the chart options. See vAxis.format for more information. The array returned will have two values, the first for the left hand axis, the columns, and the second for the right axis. Type: Array of Strings
values The numerical values of the data point. The array returned will have two values, the first for the left hand axis, the columns, and the second for the right axis. Type: Array of Numbers

Examples

Optional parameters can be passed as a single object in the class object call, these are used to style the chart and its elements.

Example 1

The following example is an object containing the data series and additional parameters, these parameters set the columns width to 70%, the top of the bars to be rounded and applies an inset shadow.

let options = {
	data: [['A', 500, 21], ['B', 436, 25], ['C', 300, 19], ['D', 375, 12]],
	columns: {
		width: 70,
		rounded: true,
		inset: true
	}
}

Example 2

In addition to passing options for the columns, there are options available for the line and points. the below example shows some of these options.

let options = {
	data: [['A', 500, 21], ['B', 436, 25], ['C', 300, 19], ['D', 375, 12], ['E', 300, 30]],
	columns: {
		width: 70,
		color: 'lightgrey',
		rounded: true
	},
	line: {
		width: 4,
		color: 'deeporange',
		pointSize: 0
	}
}

This will generate the below chart. Tip: Use the random data generator button below the chart to generate and apply a new random data set.

Example 3

The options also include vAxes, individual axis settings are seperated within as 0 and 1. 0 represents the left side axis, the bars. Each axis can be adjusted seperately such as formatting and minimum.

let options = {
	data: [['A', 500, 21], ['B', 436, 25], ['C', 300, 19], ['D', 375, 12], ['E', 300, 30]],
	columns: {
		width: 60,
		rounded: true
	},
	line: {
		width: 2,
	},
	vAxes: {
		0: {
			min: 'auto'
		},
		1: {
			format: 'time'
		}
	}
}

This will generate the below chart. Tip: Use the random data generator button below the chart to generate and apply a new random data set.

Top: Both the columns and the point have hover triggers that will show the tooltip.

Configuration

Options
data An array of arrays where each child array has three elements. The first point of the child arrays should be formatted as a string, this will be the name of the data point, the second is the numerical value for the left axis, the bars. The thrid will be a numerical value for the right axis, the line.

A maximum of 20 points can be displayed on the chart, data will be automatically truncated where the array contains more than 20 child arrays. 
options = {
	data: [
		['A', 5, 12],
		['B', 10, 14],
		['C', 15, 16],
		['D', 20, 20],
	]
}
Type: Array of Arrays Default: null
chart.title The title to appear above the chart, when not specified, the chart will expand to fill the height of the parent element automatically. Type: String Default: null
chart.color The background color of the chart. Type: String Default: white
chart.minorLines Enables or disables minor gridlines on the vertical axis. Type: Boolean Default: false
columns.width Percentage width of each columns, where 100 means there is no space between bars. This cannot exceed 100. Type: Number Default: 90
columns.color The base color of the bars, when not specified the bars will take the default array of colors.
See Pallet options for a list of available colors, these should be passed as strings. Type: String Default: multi
columns.rounded Sets the top of the bars to have rounded corners. Type: Boolean Default: false
columns.inset Sets the bars to have an inset drop shadow. Type: Boolean Default: false
columns.stripe Applies a striped overlay on the bars, this will not affect the specified color. Type: Boolean Default: false
columns.animated Animates the striped overlay of the bars, will have no effect unless columns.stripe is set to true. Type: Boolean Default: false
columns.onClick A function to be called when a columns in the chart is clicked. The function will return a single object containing elements and data related to the columns clicked, see Functions for more information. 
obj = {
	columns: HTMLElement,
	label: HTMLElement,
	name: String,
	text: String,
	value: Number
}
Type: Function Default: null
line.width Width in pixles of the series line, this value should be a whole number between 1 and 21 inclusive. Type: Number Default: 4
line.pointSize Radius in pixles of the series point markers, this value should be a whole number between 0 and 15 inclusive. When set to 0, the markers will not be visible. Type: Number Default: 6
line.color The base color of the line, when not specified the bars will take the default of indigo.
See Pallet options for a list of available colors, these should be passed as strings. Type: String Default: 'indigo
line.smooth Sets the line to be smoothed out at each series point. When set to false the lines will be straight and have shap angles at each point. A smooth line may exceed the chart bound on the curve depending on the difference and value of each series point. Type: Boolean Default: true
vAxes vAxes does not take direct parameters, each axis should have it's own parameters specifies within this object. The left axis, relating to the bars is at index 0, the right is at index 1. 
vAxes: {
	0: {
		... ...
	},
	1: {
		... ...
	}
}
vAxes.label The name of the vertical axis values, this name will show inside the tool tip along side the value for the data points. Type: String Default: Series
vAxes.lines Enables or disables horizontal major and minor gridlines on the chart for the vertical axis points Type: String Default: true
vAxes.format Formatting of the axis labels and the tool tip value label. See the Labels for a guide of the available formats. Type: String Default: number
vAxes.min Sets the minimum value of the vertical axis. When left to default, all columns will start at 0. Optionally the string value 'min' can be passed where the chart will automatically calculate the minimum value and upper value of the vAxis. Type: Number or String Default: 0
labels.position Sets the default position of the columns value labels. The position will automatically switch where the label cannot fit in the defined position. In addition, if each label will not fit within the series width, the labels will automatically hide.
  • none: labels will not be displayed
  • outside: labels will appear above the bars
  • inside: labels will appear inside the bars
Type: String Default: none
labels.color Sets the text color of the labels. See Pallet for a list of the available colors. By default, the labels will automatically switch between black and white depending on the background color. Type: String Default: match

Full Default Configuration

options = {
	data: null,
	columns: {
		width: 90,
		color: 'multi',
		rounded: false,
		inset: false,
		striped: false,
		animated: false,
		onClick: null
	},
	line: {
		color: null,
		width: 2,
		pointSize: 4,
		smooth: true
	},
	vAxes: {
		0: {
			label: 'Series 1',
			lines: true,
			format: 'number',
			min: 0
		},
		1: {
			label: 'Series 2',
			lines: false,
			format: 'number',
			min: 0
		}
	},
	labels: {
		position: 'none',
		color: 'match'
	},
	chart: {
		title: null,
		color: 'white',
		minorLines: true
	}
}
