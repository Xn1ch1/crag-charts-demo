Visualization: Column Chart

Overview

The column chart is a vertical columns chart rendered in the browser using SVG and HTML. Like all Crag Charts, column charts display tooltips when the user hovers over the series elements. The chart is also limited to 20 data points and data will automatically be truncated if there are more points provided.

The minimum required code required for creating a column chart is the following

<div id="column-chart" style="width: 100%; height: 500px;"></div>

<script src="https://combinatronics.com/Xn1ch1/Crag-Charts/main/crag/crag.charts.loader.js"></script>
<script>
	let chart;
	new CragLoader(createCharts, ['column']);

	function createCharts() {
		const data = [['A', 500], ['B', 432], ['C', 300]];
		chart = new CragColumn(data).create('column-chart');
	}
</script>

In the second script tag, the onLoadCallBack is invoked, the function paramater passed should be below this callback that will create the chart. For more detailed information, see How to Use.

Built In Functions

Title Update

The chart title can be updated at any time even if one was not defined in the initial options, should the chart need adjusting for the new space the _draw function will be called automatically. The title property should be applied to the chart object.

chart.title = 'New Chart Title';

Data Update

To update the data in the chart, the update function should be called on the chart object. Only one parameter is required, an array of arrays. See data parameter for more information. The data set can change in size when updated, columns will be automatically added and removed as needed but will still be limited to 20 points and truncated automatically. In addition, this function will automatically call the _draw function (below) once the update has been completed.

let newData = [['A', 160], ['B', 527], ['C', 428]];
chart.update(newData);

Draw

The _draw function will cause a complete redraw of the chart with the pre existing data. This function will be useful where the size of the container object changes size. The below example is calling the _draw function on the chart object when the window resizes.

window.onresize = function() {
	chart._draw();
}

On Click Callback

A function can be assigned to the onClick property of the columns elements. This function will be called when a series columns is clicked.

let options = {
	columns: {
		onClick: myFunction
	}
}
myFunction(obj) {
	... ...
}

The function should have one parameter, an object which will be returned by the click listener. This object will contain information on the series point clicked in addition to the elements related to it. The object will contain the following

Object Parameters
columns The html element of the columns clicked as an object. Type: HTMLElement
label The html element of the label related to the columns clicked as an object. If the label positions are none, this will return null. Type: HTMLElement
name The name of the data point, this is the same name used in the hAxis. Type: String
text The formatted string of the data point value. The format can be defined by the chart options. See vAxis.format for more information. Type: String
value

The numerical value of the data point.

Type: Number

Examples

Optional parameters can be passed as a single object in the class object call, these are used to style the chart and its elements.

Example 1

The following example is an object containing the data series and additional parameters, these parameters set the columns width to 90%, the top of the columns to be rounded and applies an inset shadow.

let options = {
	columns: {
		width: 80,
		rounded: true,
		inset: true
	}
}

Example 2

In this example, the options passed have included vAxis and chart settings. The two parameters passed in vAxis are min and format. See Labels for more infomration on formatting axes labels. A min setting of 'auto' will let the chart decide the lower bound value on the vertical axis.

Additionally, a chart title has also been specified and extra columns options have been passed such as color and a striped overlay.

let options = {
	columns: {
		width: 95,
		rounded: true,
		inset: true,
		striped: true,
		animated: true,
		color: 'amber'
	},
	vAxis: {
		min: 'auto',
		format: 'time'
	}
	chart: {
		title: 'Time Taken',
		color: 'darkgrey'
	}
}

Example 3

This final example demonstrates the onClick callback on the columns elements. The callback will return a single object containing information and HTML objects, see columns.onClick for more information on the object.

let options = {
	columns: {
		width: 80,
		rounded: true,
		color: 'bluegrey',
		onClick: myFunction
	},
	vAxis: {
		format: 'time'
	}
}
function myFunction(obj) {
	nameElement.textContent = obj.name;
	valueElement.textContent = obj.value;
	textElement.textContent = obj.text;
}

Configuration

Options
data An array of arrays where each child array has two elements. The first point of the child arrays should be formatted as a string, this will be the name of the data point, the second is the numerical value for this point.

A maximum of 20 points can be displayed on the chart, data will be automatically truncated where the array contains more than 20 child arrays. 

const data = [
		['A', 5],
		['B', 10],
		['C', 15],
		['D', 20],
	]
}
Type: Array of Arrays Default: null
chart.title The title to appear above the chart, when not specified, the chart will expand to fill the height of the parent element automatically. Type: String Default: null
chart.color The background color of the chart. Type: String Default: white
chart.minorLines Enables or disables minor gridlines on the vertical axis. Type: Boolean Default: false
columns.width Percentage width of each columns, where 100 means there is no space between columns. This cannot exceed 100. Type: Number Default: 90
columns.color The base color of the columns, when not specified the columns will take the default array of colors.
See Pallet options for a list of available colors, these should be passed as strings. Type: String Default: multi
columns.rounded Sets the top of the columns to have rounded corners. Type: Boolean Default: false
columns.inset Sets the columns to have an inset drop shadow. Type: Boolean Default: false
columns.stripe Applies a striped overlay on the columns, this will not affect the specified color. Type: Boolean Default: false
columns.animated Animates the striped overlay of the columns, will have no effect unless columns.stripe is set to true. Type: Boolean Default: false
columns.onClick A function to be called when a columns in the chart is clicked. The function will return a single object containing elements and data related to the columns clicked, see Functions for more information. 
obj = {
	columns: HTMLElement,
	label: HTMLElement,
	name: String,
	text: String,
	value: Number
}
Type: Function Default: null
vAxis.label The name of the vertical axis values, this name will show inside the tool tip along side the value for the data points. Type: String Default: Series
vAxis.lines Enables or disables horizontal major and minor gridlines on the chart for the vertical axis points Type: String Default: true
vAxis.format Formatting of the axis labels and the tool tip value label. See the Labels for a guide of the available formats. Type: String Default: number
vAxis.min Sets the minimum value of the vertical axis. When left to default, all columns will start at 0. Optionally the string value 'min' can be passed where the chart will automatically calculate the minimum value and upper value of the vAxis. Type: Number or String Default: 0
labels.position Sets the default position of the columns value labels. The position will automatically switch where the label cannot fit in the defined position. In addition, if each label will not fit within the series width, the labels will automatically hide.
  • none: labels will not be displayed
  • outside: labels will appear above the columns
  • inside: labels will appear inside the columns
Type: String Default: none
labels.color Sets the text color of the labels. See Pallet for a list of the available colors. By default, the labels will automatically switch between black and white depending on the background color. Type: String Default: match

Full Default Configuration

options = {
	columns: {
		width: 90,
		color: 'multi',
		rounded: false,
		inset: false,
		striped: false,
		animated: false,
		onClick: null
	},
	vAxis: {
		label: 'Series',
		lines: true,
		format: 'number',
		min: 0
	},
	labels: {
		position: 'none',
		color: 'match'
	},
	chart: {
		title: null,
		color: 'white',
		minorLines: true
	}
}
