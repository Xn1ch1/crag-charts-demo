Visualization: Progress Chart

Overview

The progress chart is a single horizontal columns chart rendered in the browser using SVG and HTML, the columns represents the progress of a single data point against a total with an optional mid point target. The chart takes as minimum one value passed in the class options and requires a parent with a minimum of 250px in height.

The below if the minimum code required for a chart

<div id="progress-chart" style="width: 100%; height: 300px;"></div>

<script src="https://combinatronics.com/Xn1ch1/Crag-Charts/main/crag/crag.charts.loader.js"></script>
<script>
	let chart;
	new CragLoader(createCharts, ['progress']);

	function createCharts() {
		const options = {
			current: {
				value: 43
			}
		}
		chart = new CragProgress(options).create('progress-chart');
	}
</script>

Built In Functions

Draw

The _draw function will cause a complete redraw of the chart with the pre existing data. This function will be useful where the size of the container object changes size. The below example is calling the _draw function on the chart object when the window resizes.

window.onresize = function() {
	chart._draw();
}

Updating Values

Once the chart has been created, the current target and total value can be updating using the setter functions for each on the chart object. Calling these will cause an automatic redraw of the chart.

const options = {
	current: {
		value: 10
	},
	target: {
		value: 45
	}
}

let chart = new CragProgress(options).create('progress-chart');

function updateCurrent(val)
	chart.current = val;
}

function updateTotal(val)
	chart.total = val;
}

function updateTarget(val)
	chart.target = val;
}

Examples

Optional parameters can be passed as a single object in the class object call, these are used to style the chart and its elements.

Example 1

In the following example, the columns has been stylized with and inset shadow and animated stipes. In addition, a color has been specified for each of the values, this will automatically change the progress columns color when the target is reached and then the total.

const options = {
	columns: {
		inset: true,
		striped: true,
		animated: true
	},
	current: {
		value: 25,
		color: 'red'
	},
	target: {
		value: 50,
		color: 'amber'
	}
	total: {
		color: 'green'
	}
}

Example 2

In this example, the chart has been given a title, the target, the total has been removed and the current value has been set to percentage and inside. This moves the current value to the center of the columns and removed the indicator line.

const options = {
	chart: {
		title: 'Current Progress'
	},
	current: {
		value: 72,
		color: 'bluegrey',
		percentage: true,
		inside: true
	},
	total: {
		show: false
	}
}

Configuration

Configuration Options
chart.title The title to appear above the chart, when specified, the columns will move down from the center of the parent to allow for the title margins. Type: String Default: null
chart.color The background color of the chart. Type: String Default: 'white'
columns.inset Sets the progress columns container to have an inset drop shadow. Type: Boolean Default: false
columns.stripe Applies a striped overlay on the progress columns, this will not affect the specified color. Type: Boolean Default: false
columns.animated Animates the striped overlay of the columns, will have no effect unless striped is set to true. Type: Boolean Default: false
current.value Sets the initial value of the progress columns. This number can not exceed total. Type: Number Default: 0
current.label Sets the text content displayed for the label below the current value. This is only visible when current.inside = false and current.show = true. Type: String Default: 'Current'
current.color Sets the base color of the progress columns. Type: String Default: 'blue'
current.percentage Sets the current value to show as a percentage instead of the default number. Type: Boolean Default: false
current.inside Sets the curent value to show inside the progress columns. This will also remove the label and the progress indicator. Type: Boolean Default: false
current.show Sets the current value, value label and progress indicator line visible or invisible. Type: Boolean Default: true
current.value Sets the initial value of the progress columns. This number can not exceed total. Type: Number Default: 0
target.label Sets the text content displayed for the label above the target value. This is only visible when the target is set to greater than 0. Type: String Default: 'Target'
target.color Sets the color color of the progress columns once the target has been met. Type: String Default: 'blue'
total.value Sets the total value of the progress columns. This number can not be lower than target or current. Type: Number Default: 0
total.label Sets the text content displayed for the label above the total value. This is only visible when total.show = true. In addition this will also self hide where the target is close enough to overlap the value or label. Type: String Default: 'Total'
total.color Sets the color of the progress columns when the total has been reached. When not set the color will remain its curent or target color where specified. Type: String Default: 'blue'
total.show Sets the toal value, value label and progress indicator line visible or invisible. Type: Boolean Default: true
