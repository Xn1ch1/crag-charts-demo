Labels

Overview

Labels on charts covers vertical axis, horizonal axis and data point labels. Each visualisation can have its labels formatted using the optional parameters, these parameters must be formatted as string such as 'number'.

const options = {
	...
	vAxis: {
		format: 'time'
	},
	...
}

In addition to formatting, labels can be given a color. By default, label colors will automatically be determined by the background color of the containing object, the two possible colors are white or almostblack.

Configuration

Options
number Formats values as strings with thousand separators. This format uses the clients locale to determine the currect seperator to use.
  • 4321 = '4,321'
  • 0030 = '30'
decimal Formats values as strings with thousand separators and two decimal places. This format uses the clients locale to determine the currect seperator to use.
time Formats values, taken in seconds, as time.
  • 752 = ‘12:32'
  • 9019 = ‘02:30:19'
