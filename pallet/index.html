Color Pallet

Overview

The library has a built in color pallet, this is to standardize the look and feel of charts. All chart options that take a color value should be passed as a string, for example 'teal'.

options = {
	chart: {
		color: 'white'
	},
	bars: {
		color: 'teal'
	}
}

By default, all chart labels will be in match mode, this mode will set the labels between white and almostblack depending on the color of the object they are inside.
