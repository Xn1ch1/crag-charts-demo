Crag Charts

Overview

Crag Charts is light-weight JavaScript library for modern browsers designed to make the implementation of charts into existing and new websites a simple and easy affair. The main ethos and goal of Crag Charts is minimal code whilst maintaining high customisation and scalability.

The library was created from the need for a JavaScript chart library, without the burden of expensive and heavy polyfills supporting long outdated and disused browsers. The result is a lightweight and speedy library with a modern look.

Charts are rendered in a mixture of HTML elements and SVG where appropriate.
